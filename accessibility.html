Accessibility Statement

Daily Birthday Song aims to make the website accessible, clear and comfortable for as many users as possible.

1. Accessibility approach

The website was built with attention to readable text, simple navigation, clear buttons and responsive display for desktop, tablet and mobile devices.

2. Accessibility tools

The website includes an accessibility button that allows users to activate options such as dark mode, large text, readable font, bold text, high contrast and highlighted links.

3. Keyboard and navigation

We aim to keep navigation simple and accessible. Links and buttons are designed to be clear and easy to identify.

4. Media content

The website contains music videos and visual content. Some media may depend on the browser, device and external technical limitations.

5. Ongoing improvements

Accessibility is an ongoing process. We continue to improve the website and fix issues when they are found.

6. Accessibility feedback

If you experienced an accessibility issue, difficulty using the website or a missing accessibility feature, please contact us through the email or WhatsApp details on the website. Please include a short description of the issue, the page address and the device or browser you used.
